# Don't Reneg On Me 🃏

> Say Less, Play More.

A premium Spades card game with real-money tournament features.

## Stack
- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Deploy: Vercel (`spades-royale.vercel.app`)

## Game Features
- Full Spades engine (13 tricks, 4 players, score to 500)
- Strict reneg detection & enforcement
- Nil / Blind Nil bidding
- Sandbag penalty system (10 bags = -100)
- AI opponents with bidding logic
- Premium BBQ/graffiti aesthetic

## Dev
```bash
pnpm install
pnpm dev
pnpm build
```
